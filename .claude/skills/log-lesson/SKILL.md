# Log Lesson Skill

When corrected:
1. Add lesson/rule to LEARNINGS.md
2. Add current-state note to daily memory file
3. Keep entries concise, factual, and actionable
4. Include what changed and why

If a correction is high impact, promote it to a hard rule in project docs.
