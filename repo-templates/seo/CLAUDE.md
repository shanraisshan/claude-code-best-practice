# CLAUDE.md — SEO

## Project Overview
SEO-focused repository for technical audits, content optimization, structured data, and search performance. Claude Code is configured with SEO-specific skills.

## What This Repo Covers
- Technical SEO audits and fixes
- Schema markup / structured data (JSON-LD)
- Content strategy and keyword planning
- Programmatic SEO page generation
- AI search optimization (AEO/GEO)
- Analytics and tracking setup

## Conventions
- Audit reports go in `audits/`
- Schema templates in `schemas/`
- Programmatic page templates in `templates/`
- Analytics configs in `tracking/`

## Skills Available (~7)
- **seo-audit** — Technical SEO audits, crawl errors, indexing, Core Web Vitals
- **schema-markup** — JSON-LD structured data, rich snippets, schema.org
- **ai-seo** — Optimize for AI search engines (AEO, GEO, LLMO)
- **programmatic-seo** — Template-based pages at scale (directories, location pages)
- **content-strategy** — Topic planning, content calendar, keyword strategy
- **analytics-tracking** — GA4, conversion tracking, UTM, event tracking
- **competitor-alternatives** — Competitor comparison and "vs" pages

## Additional Skills (add if needed)
- **site-architecture** — URL structure, internal linking, navigation hierarchy

## Workflow
1. Start with `seo-audit` to identify issues
2. Use `schema-markup` to fix structured data
3. Use `content-strategy` to plan new content
4. Use `programmatic-seo` for scalable page creation
5. Set up `analytics-tracking` to measure results

## Guidelines
- Always check for `product-marketing-context.md` before auditing
- Keep this file under 80 lines for your actual project
