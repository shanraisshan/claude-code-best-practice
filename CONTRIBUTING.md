# Contributing

Contributions are welcome — tips, workflows, fixes, and new sections all help.

## What to contribute

| Type | Examples |
|------|---------|
| **Tips** | Prompting patterns, context management tricks, session shortcuts |
| **Workflows** | Development methodologies used in real projects |
| **Fixes** | Broken links, outdated feature names, version corrections |
| **New sections** | Troubleshooting guides, cost optimization, migration notes |
| **Skills / Agents** | Links to curated collections worth adding |

## How to contribute

1. Fork this repo
2. Create a branch: `git checkout -b your-change-name`
3. Make your changes
4. Open a pull request with a clear title and a one-line description of what changed and why

## Style guide

- Match the existing table format in README.md for new feature entries
- Use attribution — if a tip comes from a specific person or source, credit them with a link
- Keep descriptions concise: one sentence per feature, two max
- No marketing language — describe what a feature does, not how great it is
- If adding a tip, note which version it applies to if version-specific

## Reporting issues

Open an issue for:
- Broken links
- Outdated information (feature removed, flag renamed, etc.)
- Missing content you'd like to see but can't contribute yourself

## Questions

Open an issue or start a discussion. The maintainer (@shanraisshan) is active.
